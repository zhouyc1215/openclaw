## 🔧 常用命令

```bash
# 查看任务
openclaw cron list

# 手动触发
openclaw cron run <task-id>

# 查看日志
tail -f /tmp/openclaw-gateway.log

# 重启 Gateway
openclaw gateway restart

# 禁用任务
openclaw cron disable <task-name>

# 启用任务
openclaw cron enable <task-name>
```

## 🐛 故障排查

### 任务未执行？

```bash
ss -ltnp | grep 18789  # 检查 Gateway
openclaw cron list     # 查看任务状态
```

### 消息未送达？

检查：

1. Gateway 运行
2. 飞书配置
3. 用户 ID

### 任务超时？

编辑 `~/.openclaw/cron/jobs.json`：

```json
"payload": {
  "timeoutSeconds": 300
}
```
