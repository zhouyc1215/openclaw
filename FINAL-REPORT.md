## 🔧 技术细节

### 配置文件位置

- **Cron 任务**: `~/.openclaw/cron/jobs.json`
- **OpenClaw 配置**: `~/.openclaw/openclaw.json`
- **Gateway 日志**: `/tmp/openclaw-gateway.log`

### 任务配置结构

```json
{
  "id": "uuid",
  "agentId": "main",
  "name": "task_name",
  "enabled": true,
  "schedule": {
    "kind": "cron",
    "expr": "0 8 * * *",
    "tz": "Asia/Shanghai"
  },
  "sessionTarget": "isolated",
  "wakeMode": "now",
  "payload": {
    "kind": "agentTurn",
    "message": "prompt",
    "timeoutSeconds": 240
  },
  "delivery": {
    "mode": "announce",
    "channel": "feishu",
    "to": "ou_b3afb7d2133e4d689be523fc48f3d2b3",
    "bestEffort": true
  }
}
```

### 关键参数说明

- **timeoutSeconds**: 180-240 秒（根据任务复杂度）
- **bestEffort**: true（失败不发送错误消息）
- **sessionTarget**: isolated（独立会话）
- **wakeMode**: now（立即执行）
